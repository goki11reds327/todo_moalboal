## How To Environment

<p>
 Step 1. git clone URL<br>
 Step 2. DB作成<br>
 Step 3. cp .env.sample .envを入力<br>
 Step 4. php key:generateを入力し、Keyを作成<br>
 Step 5. php artisan migrateでテーブル作成<br>
 Step 6. php artisan serve<br>
</p>
